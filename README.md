# Empathic Legal

A guided experience for reaching agreements without confrontation.

Built with React + TypeScript + Vite + Supabase.

## Development

```bash
bun install
bun run dev
```

## Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase (auth + database + edge functions)
- Anthropic Claude API (AI mediation)
- react-i18next (Hebrew primary, English fallback)
- Heebo font (RTL-ready)
